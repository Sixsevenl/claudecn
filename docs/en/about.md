# About Claude Chinese Community

Claude Chinese Community is a non-profit technical community dedicated to promoting Claude AI technology. We provide comprehensive Chinese documentation, tutorials, and best practices to help developers better use Claude's various features.

## Content Coverage

- **Claude Code**: Anthropic's AI programming assistant with terminal and IDE integration
- **Agent Skills**: Skill system for building intelligent agents
- **Everything Claude Code**: AI Agent performance optimization system
- **Extended Thinking**: Chain-of-Thought reasoning techniques
- **Computer Use**: Computer Use automation capabilities
- **MCP Integration**: Model Context Protocol tool connections

## Community Contribution

All developers interested in Claude AI are welcome to contribute! You can participate by:

1. Submitting documentation improvement suggestions
2. Sharing usage experiences and best practices
3. Translating English documentation
4. Reporting issues and bugs
