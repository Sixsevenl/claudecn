# 关于 Claude 中文社区

Claude 中文社区是一个致力于推广 Claude AI 技术的非营利性技术社区。我们提供全面的中文文档、教程和最佳实践，帮助开发者更好地使用 Claude 的各项功能。

## 内容范围

- **Claude Code**：Anthropic 推出的 AI 编程助手，支持终端和 IDE 集成
- **Agent Skills**：构建智能 Agent 的技能系统
- **Everything Claude Code**：AI Agent 性能优化系统
- **扩展思维**：Chain-of-Thought 推理技巧
- **计算机控制**：Computer Use 自动化能力
- **MCP 集成**：Model Context Protocol 工具连接

## 社区贡献

欢迎所有对 Claude AI 感兴趣的开发者参与贡献！你可以通过以下方式参与：

1. 提交文档改进建议
2. 分享使用经验和最佳实践
3. 翻译英文文档
4. 报告问题和 Bug
